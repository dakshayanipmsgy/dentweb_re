# Play Store Listing Draft

## App name

Dakshayani

## Short description

Secure Dakshayani workspace access for admins, employees, and registered customers.

## Full description

Dakshayani provides secure mobile access to the Dakshayani Enterprises workspace for approved users. Admins, employees, and registered customers can sign in to their role-based dashboards and access the workflows available to their account.

The app is designed as a secure web-based workspace. Internet access is required for login, dashboards, documents, and support workflows.

Key features may include:

- Secure login for authorized users.
- Admin dashboard access.
- Employee workspace access.
- Customer dashboard access.
- Quotation, agreement, dispatch advice, challan, invoice, and payment receipt access where available to the signed-in user.
- Complaints and support access.
- Task, lead, and customer workflows where applicable to the user's role and permissions.
- Mobile-friendly navigation for quick access to common workspace areas.

Dakshayani protects private workspace access through authenticated role-based pages. Users should only sign in with accounts issued or approved by Dakshayani Enterprises.
